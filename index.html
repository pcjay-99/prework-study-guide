HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Header tags are used in sequence: H1-H6, and H1 can only be used once
  • Semantic elements describe it's contecnts to both the user and the browser. Use first.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Three ways to style using CSS: inline CSS, internal CSS, and external CSS
  • External CSS stylesheets are more efficient

Git

  • git status: checks what branch we are currenlty on
  • git checckout -b branch-namne: creates a new branch and switches to it
  • git pull will pull down the branch named

Javascript

  • A variable is a named container tht allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Function(s) are a set of instructions that tell a computer how to do a certain task.
  • Functions must be called, they do no run automaticlly
  • Math.floor and Math.random can be used together to generate a random number
